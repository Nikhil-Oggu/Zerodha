# Zerodha Clone

A full-stack Zerodha-like trading platform built using **React**, **Node.js**, **Express**, and **MongoDB**.  
This project includes authentication, protected dashboard routes, order placement, holdings, positions, and a watchlist.



##  Live Demo

**Frontend (Render Static Site)**  
https://zerodhaclone-ldn1.onrender.com/

**Backend API (Render Web Service)**  
https://zerodhaclonebackend-8bbw.onrender.com

## Tech Stack

### Frontend
- React 18
- React Router v6
- Context API
- CSS 
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcryptjs

### Deployment
- Render (Static Site for frontend)
- Render (Web Service for backend)
- MongoDB Atlas


##  Features

- User Signup & Login (JWT based)
- Protected Dashboard Routes
- Watchlist
- Buy / Sell Orders
- Holdings & Positions
- Funds Summary
- Persistent Authentication
- Responsive UI

##  Authentication Flow

1. User logs in / signs up
2. JWT token generated by backend
3. Token stored in `localStorage`
4. Protected routes verified using `ProtectedRoute`
5. Token validated via `/verify` API
